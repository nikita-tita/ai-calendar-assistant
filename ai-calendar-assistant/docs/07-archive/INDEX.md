# Архив документации

Этот раздел содержит устаревшие документы для справки и истории.

## Правила архивации

1. Документы перемещаются в папку соответствующего квартала
2. К названию добавляется префикс `[ARCHIVED_YYYY-MM-DD]_`
3. В начало документа добавляется заголовок архива
4. Запись добавляется в этот индекс

## Как архивировать документ

```bash
# 1. Определите квартал
QUARTER="2025-Q1"
DATE=$(date +%Y-%m-%d)

# 2. Создайте папку квартала
mkdir -p docs/07-archive/$QUARTER

# 3. Переместите с префиксом
mv docs/путь/к/DOCUMENT.md \
   docs/07-archive/$QUARTER/[ARCHIVED_$DATE]_DOCUMENT.md

# 4. Добавьте заголовок архива в документ
echo "# [АРХИВ] Название

⚠️ **ЭТОТ ДОКУМЕНТ УСТАРЕЛ**

**Дата архивации**: $DATE
**Причина**: Описание причины
**Замена**: [Новый документ](../../путь/к/новому.md)

---
" | cat - docs/07-archive/$QUARTER/[ARCHIVED_$DATE]_DOCUMENT.md > temp && \
mv temp docs/07-archive/$QUARTER/[ARCHIVED_$DATE]_DOCUMENT.md

# 5. Добавьте запись в этот индекс
```
