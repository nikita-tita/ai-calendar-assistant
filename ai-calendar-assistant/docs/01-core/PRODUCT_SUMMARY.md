## 📞 Техническая поддержка

### Мониторинг:
```bash
# Статус сервисов
ssh root@91.229.8.221 "docker ps"

# Логи бота
ssh root@91.229.8.221 "docker logs -f telegram-bot"

# Health check
curl https://xn--80aiabdqfcqhgchaebm7bp0qg3a.xn--p1ai/api/health
```

### В случае проблем:
1. **Бот не отвечает:**
   - Проверить Docker: `docker ps`
   - Перезапустить: `docker restart telegram-bot`
   - Проверить логи: `docker logs telegram-bot`

2. **WebApp не открывается:**
   - Проверить Nginx: `systemctl status nginx`
   - Проверить SSL: `certbot certificates`
   - Проверить DNS: `dig этонесамыйдлинныйдомен.рф`

3. **Медленная работа:**
   - Проверить ресурсы: `htop`
   - Проверить логи: `tail -f /var/log/nginx/error.log`
   - Масштабировать VPS
