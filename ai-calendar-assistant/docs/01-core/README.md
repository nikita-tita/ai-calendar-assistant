# 01-core: Основная документация проекта

Эта папка содержит фундаментальную документацию проекта AI Calendar Assistant.

## Содержание

- [ARCHITECTURE.md](ARCHITECTURE.md) - Архитектура системы, компоненты, взаимодействие
- [PROJECT_SUMMARY.md](PROJECT_SUMMARY.md) - Краткое описание проекта, цели, задачи
- [PRODUCT_SUMMARY.md](PRODUCT_SUMMARY.md) - Продуктовое описание, ценность для пользователей
- [TODO.md](TODO.md) - Текущие задачи и планы развития
- [DEVELOPMENT.md](DEVELOPMENT.md) - Руководство для разработчиков

## Для новых разработчиков

Рекомендуемый порядок изучения:

1. **Начните с** [PROJECT_SUMMARY.md](PROJECT_SUMMARY.md) - поймите что делает проект
2. **Изучите** [ARCHITECTURE.md](ARCHITECTURE.md) - разберитесь в архитектуре
3. **Прочитайте** [DEVELOPMENT.md](DEVELOPMENT.md) - узнайте как разрабатывать
4. **Посмотрите** [TODO.md](TODO.md) - ознакомьтесь с планами

## Диаграммы

Папка `diagrams/` содержит визуальные схемы архитектуры проекта.
