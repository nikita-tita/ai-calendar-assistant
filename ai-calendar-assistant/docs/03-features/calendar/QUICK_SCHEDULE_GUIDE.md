# Быстрое Создание Расписания

## Как создать несколько событий за раз

Теперь вы можете отправить боту целое расписание на день, и он создаст все события автоматически!

### Формат

Просто напишите список событий с временем:

```
тайминг на завтра:
09:00-10:00 Встреча с командой
10:00-11:30 Обзор проекта
11:30-12:00 Кофе-брейк
12:00-13:00 Презентация
13:00-14:00 Обед
14:00-16:00 Работа над задачами
```

### Правила

1. **Минимум 3 события** - нужно хотя бы 3 строки с временем
2. **Формат времени:** `ЧЧ:ММ-ЧЧ:ММ Название` или просто `ЧЧ:ММ Название`
3. **Дата:** укажите "на завтра", "на 23 октября", или другую дату

### Примеры

**С диапазоном времени:**
```
расписание на сегодня:
12:45-13:00 Приезд
13:00-13:30 Кофе-брейк
13:30-15:00 Дискуссия
```

**С одним временем (длительность 1 час):**
```
план на завтра:
09:00 Утренняя планерка
11:00 Созвон с клиентом
15:00 Ревью кода
```

**Ключевые слова** (необязательно, но помогают):
- тайминг
- расписание
- план
- schedule
- agenda

### Что произойдет

1. Бот покажет сколько событий создаст
2. Вы увидите кнопки [Подтвердить] [Отменить]
3. После подтверждения все события появятся в календаре

---

**Примечание:** Все события будут созданы на указанную дату с точным временем начала и окончания.
